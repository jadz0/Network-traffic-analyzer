# Analysis Functions

## Protocol Distribution

Purpose:

Identify the types of protocols observed within the capture.

Protocols:

* TCP
* UDP
* ICMP

---

## Top Source IP Analysis

Purpose:

Identify the most active source hosts within the capture.

Use Cases:

* Traffic profiling
* Host activity analysis
* Network visibility

---

## Top Destination IP Analysis

Purpose:

Identify the most contacted destinations within the capture.

Use Cases:

* Network investigation
* External communication analysis
* Traffic monitoring

---

## Traffic Reporting

Generate CSV reports for documentation and future analysis.
